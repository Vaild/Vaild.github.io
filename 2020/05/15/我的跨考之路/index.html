我的跨考之路

发表于 2020-05-15 更新于 2020-05-18 分类于 RecordLife Disqus：

要有敢于选择地勇气与决心

海洋资源开发技术跨考计算机技术

初试第一，录取成绩第一 总分：400 英语：67 政治：66 数学二：128 数据结构(909)：139

我为什么会选择跨考

最重要的一个原因，我想用比较官方的说法我考研复试的答案来讲一下。

计算机在想其他方向融合的趋势势不可挡，计算机在未来的工作的选择面肯定会更广。

我其实从大三刚开始就已经有跨考的想法了，这个想法中间变过很多次，比如在大三的寒假里，我开始想考海大、中科院海洋研究所等等，都是出于如果我考不上会怎么样的想法，还有如果我学不懂《数据结构》会怎么样的想法，最终是理性的胜利。其实做这个决定真的是很难，但我最终还是走上了这条路，不是吗？

这一路走来，经历的很多，也有很多的感慨，但我认为这都是值得的，而且在这个过程中我确实也成熟了不少。

我认为比较重要的几件事

  • 跨考之前要想好，能不能面对压力。毕竟开弓没有回头箭。

  • 前期注意劳逸结合，心态很重要，所以我在9月中旬之前一直是每周休一天，每隔一天去游个泳(因为游泳馆每隔一天免费)。

  • 有个研友相互鼓励，我是和室友。室友不和我考一个专业也不一个学校，没什么压力。只不过有个室友，在后来放弃了。

  • 单词背诵贯穿整个学习过程。

  • 考研信息的收集也是一个比较重要的步骤，我是在晚上回宿舍的时间完成。

我是怎么准备的

在这里首先讲一下大概的时间脉络，再分科目来讲。

我是从2019-02-21正式决定跨考的，当时的目标按顺序大概有三个：北京理工大学、山东大学本部还有就是本校山东大学(威海)。但是，由于后来的实习，以及怕专业课变为408的惶恐，我最终还是选择了最稳妥的选项自己的学校山东大学（威海）就在这天下午，我就把C语言整本书的代码又敲了一遍，算是复习的正式开始。

  • 2019-02-21~2019-05-01：这段时间我是边上课边复习，我复习的科目主要是数学和数据结构，英语以背单词为主顺便做了《阅读的逻辑》。
  • 2019-05-13-2019-07-05：专业实习在公司搬垃圾，没有复习，一天100个单词。
  • 2019-07-13~2019-12-22：复习直到考研结束。这个时间点是可以根据学习时间接着具体细分一下的
    • 2019-07-13~2019-09-01：隔天早上去游个泳，8:30-11:50，14:30-17:50， 18:30-21:00大概，每周休一天
    • 2019-09-02~2019-09-30：7:30-11:50,14:00-17:50,18:30-22:30
    • 2019-10-02~2019-12-20：7:10-11:50,12:30-17:50,18:30-22:30,23:30-00:30/01:00

数据结构

  • 专业课用到的资料：王道的数据结构 + 前哥的资料
  • 山大的题是有一些规律的，每年知识点差不多，代码题会有去年的一道原题。

我是从一开始决定跨考就开始复习的，这当中在实习之前我是看到了图的部分，真的而是看到崩溃了，有时候就开始怀疑自己的是不是真的不是这块料不适合学这个专业，尤其中间学习KMP算法的时候，是真的崩溃了。所以在考研的过程中心态是非常重要的一点，还好我及时调整了过来。

理论部分

最重要的一件事就是啃书，就是看书我觉得看书所获得的理解之后的知识是远比视频的所得到的知识印象更加的深刻。就这样，在实习后到考研，王道的那本数据结构我至少i啃了六遍。起初的第一二遍，忘的会较快，注意结合做课后题复习，将之后带入到实践。后来的每一遍都会发现新的知识，都会往自己的知识体系当中去补充，每一遍的印象都会更加的深刻。对于题目的理解会更加的深刻。

哦，对。我是买了王道的课的，但是看的不太多，主要看一些课后题的解释。

用前哥资料的真题，再次巩固。

代码部分

王道的代码题刚开始是没有做的，后来才开始，做了有四遍吧，但是默写的就比较多了，刚开始老是忘。

《数据结构十九讲》时候来用来背了背上面的代码。20年的真题部分是我用的最多的，因为我做了三遍，上面的代码题写了少说得有几十遍吧，我是每隔几天就把我做过的写一遍，后来发现全默写一天得一个小时，我就只默写试题改革之后的部分。

我代码还有对代码理解的部分增长最快的时期应该就是在国庆节之后，我开始中午不午睡去背代码的那段时间，这个真的是深有体会，就那段时间之后，我突然觉得我是真正的理解了代码其中的含义。之前做王道代码题总是不自信去看答案，之后真的而大有改观，我看题直接按自己思路写几乎跟答案相同。这个时候才开始感觉有些入了门。

11月初，前哥的资料出了预测题，买来之后，开始做了三遍吧。

真题和预测题的最后一遍是模拟，大概一个半到两个小时做完。

在之后，把这些资料中所有出现的代码题按专题抄了题目，比如：线性表的放到一起、树的题目放到一起等。有空就默写一下，直到最后全部掌握。

  • 最后考试的时候，有前一年的一个原题，前哥的预测题好像也有一个，还有一个我的思路比较复杂实际只要递归就可以。

数学

  • 高数十八讲 + 李永乐线代 + 张宇闭关修炼 + 张宇1000题 + 张宇的真题(卷子的形式) + 张宇、李林、汤家凤预测卷
  • 个人认为：高数适合用完整成块的时间学习
  • 学习后面知识的时候，一定要回过头来做做之前的题目，否则真的会忘

这也是我下功夫比较多的，刚开始是一下午都来复习，后来开始做真题就早上用真题模拟，再到后来用晚上回宿舍之后的一段时间用来做了一遍1000题。

高数十八讲还有线性代数做了大概三遍，第一遍是在实习之前，第二遍是实习之后，在之后是用晚上的时间大概看了一遍，典型题做一下。

闭关修炼，我只做了高数部分，基本做到全部搞懂了，有些证明题是每天做一遍。我应该是在这个时候把线性代数落下的，再后来也没有时间补，只能用回宿舍的一小部分时间补。

紧接着大概还有50天的时候，开始做真题，早上8：30-11：30模拟，之后进行错题的总结，前些年的卷子一般比较简单但是也不要骄傲。我一般就是做两个小时到两个半小时的时候比较多。我是在这个时候发现自己线代基础薄弱的，开始补。这个时候一定要好好总结错题，认认真真弄明白每一个真题，不要只要求作对就可以了（有可能下次做你就发现你不会了）。

真题做完，就是模拟题，我做的顺序是：张宇、汤家凤、李林。张宇的题是真的难，汤家凤的中规中矩有些极简单，李林的题虽然比较简单，但是我认为是最有真题风格，就是那种由问题来解决问题的那种风格。这里主要还是心态问题吧，毕竟只是模拟不是真题，既不要气馁（做张宇的题有时候真的可能会崩溃）也不要过于高兴。在这期间我又看了看之前的笔记，线性代数是在晚上的回宿舍之后硬补的。

还剩几天，我就拿出来错题本还有笔记本，再看看放松一下。

英语

  • 墨墨背单词 + 阅读的逻辑 + 何凯文的长难句 + 王江涛作文 + 潘赟作文 + 张剑黄皮书
  • 多读多做才是真的。

从一打算跨考就开始背单词，在实习前把《阅读的逻辑》做完了，但是这些逻辑却没有会。

长难句是在实习之后到开学前在看，虽然我觉得而对我帮助不大。

后来，我单词用的是张剑黄皮书送的那一下本。

英语二的真题比较少，我先做了以下英语一的，后来再把英语二的真题做了两遍。做的时候，要弄懂美哟i个句子还是有些费时间的，所以我并没有采取这种策略而是对完答案就把错题所对应的句子摘抄一下，读背。每天早上在图书馆开馆之前就读前一天的文章。刚开始是每天做两篇，后来感觉差不多了，直接把后面的四套卷来做模拟。

我觉得完形填空，刚开始做的时候就可以带上，培养语感，找到有些套路，有时候就很好填词，总比放弃要好。

作文，我开始的很晚，离考试还有一个月的时候开始背的，之前背过几篇忘得差不多了，后来我也发现背不完了，我就找了潘赟的网课，开始套模板，我之后的每天写一篇，都是套模板写，写的很快，大小作文一共20多分钟到半个小时就能写完。（英语二的作文还是简单，不要学我，早开始比较好）

后来堆起来就越来越轻松，也逐渐掌握也一些小的技巧，所以做英语二的时候，后面几套几乎就是阅读扣个4分左右，但是考试的时候是真的困，有些逻辑混乱。

政治

  • 肖秀荣全套 + 徐涛的网课 + 小白考研的小程序
  • 关键靠思想觉悟背，该背得时候别偷懒
  • 我应该是九月中旬开始的

我是属于脑子比较笨得人，背东西很慢，比如最后背肖四的时候，我是中午背，整个晚上背，还是没有我室友晚上只背两个多小时背的快，最后我选择47总分66，我室友总分86…86

刚开始的基础知识肯定是要看网课的，徐涛讲课很有意思，而且确实可以学到东西，一天一章。一定要记住听完课要做题(铅笔)巩固。而且政治这个东西忘得很快，一定要及时复习。

听完课之后再去做一遍题，把不会的部分去复习一遍。我选择题部分后来用小白考研的小程序刷了半遍，可以利用吃饭的时候刷。

最后，肖八肖四，肖八我是把所有的选择题的知识点背下，简答题只做了马原的第一个，判断什么原理那个题。肖四选择弄明白，直接背简答。

虽然，今年肖大大没怎么发挥好，但是那些材料基本都对，也算是有所准备。

复试

  • 计算机网络(谢希仁)

因为疫情的原因，复试拖了很久，这也给复试足够的充分的时间进行准备，但是也产生了足够的焦虑，毕竟在家里复习的效率真的没有学校那么高。

我把课本背了三遍，第一遍没有做题只是为了能够清楚的了解这门课程，为了效率更高，我选择在楼下背书。第二遍，先背再做题（课后题+王道题），第三遍是自己在纸上写本章大概的知识点，在和书上比对进行补充。

致谢

感谢父母能够支持理解我当初的决定，以及在这一路上给与我的经济与精神支持。

感谢室友每天的欢歌笑语与调侃，让我一直保有一个好的心态。

感谢**(几个研友就不写真实姓名了)等等这一路走来的所有人。

感谢图书馆的同桌的女生(还不知道叫什么名字)，真的帮了我特别多。

谢谢！

完成之际拟录取的名单还没有已经出来！初试第一，录取成绩第一。

也有人问过我，是不是后悔当时没有报本部？

其实是有些小小的后悔的，但这个选择都是基于我当时自身情况最为合理的选择，也没有什么不好。

要有敢于选择的勇气与决心！
